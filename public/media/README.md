# Replaceable media

Keep these filenames when you replace the placeholder media with original studio work.

- `hero-residence.jpg` - first screen architectural image
- `studio-interior.jpg` - studio/interior feature poster
- `interior-film.mp4` - interior motion feature video
- `duchateau-01.jpg` to `duchateau-03.jpg` - DUCHATEAU-style horizontal portfolio placeholders
- `project-01.jpg` to `project-06.jpg` - selected project images

The React app reads these files from `/media/...`, so you can replace the assets without changing code.
